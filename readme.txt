Makes the "Visit Site" link always visible, and also open the link in a new tab/window.

== Description ==
Better Visit Site Link is a plugin that places a new visit site link to your WordPress admin bar that is always visible and also open in a new tab.

A very simple and basic plugin, but still one of the most popular plugins among our clients.

Enjoy the better visit site link.

Feel free to contact us if you have any questions about the plugin: [Your WP Expert](http://www.yourwpexpert.com/contact/)

== Installation ==
1. Upload the `better-visit-site-link` folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Enjoy a better visit site link.
